# Manual Automation App

Low-Code/No-Code платформа автоматизации тестирования на базе React и Playwright.

## Структура проекта

```
/frontend - React приложение (Vite + TailwindCSS)
/backend  - Node.js сервер (Express + Playwright)
```

## Установка и запуск

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend будет доступен на `http://localhost:5173`

### Backend

```bash
cd backend
npm install
npm start
```

Backend будет доступен на `http://localhost:3001`

## Технологии

### Frontend
- React (Vite)
- TailwindCSS
- Zustand (state management)
- @hello-pangea/dnd (drag-and-drop)
- lucide-react (иконки)
- recharts (графики)

### Backend
- Node.js
- Express
- Playwright (автоматизация браузера)
- CORS
